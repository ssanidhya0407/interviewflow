# InterviewFlow Frontend

This is the Next.js frontend for InterviewFlow.

## Quick Start

```bash
npm install
npm run dev
```

Runs on `http://localhost:3000`.

## Environment

Create `frontend/.env.local`:

```env
NEXT_PUBLIC_API_URL=http://127.0.0.1:8000
```
